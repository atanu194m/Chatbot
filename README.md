# Chatbot

A simple Python chatbot using NLTK and TensorFlow.

## Files included:
- `intents.json`: Contains the training data (patterns and responses).
- `main.py`: Trains the neural network model and generates `chatbot_model.h5`, `classes.pkl`, and `words.pkl`.
- `chatbot.py`: Runs the trained chatbot to interact with users in the terminal.
